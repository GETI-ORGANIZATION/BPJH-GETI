# VLM-Guided Attention Supervision for Multi-Instance Generation

## Idea Description
A training-free method where a Large Multimodal Model (LMM) acts as an external critic to dynamically adjust cross-attention maps in diffusion models. Instead of fixed masks, the method uses the LMM to identify semantic leakage or missing instances in intermediate steps and applies a gradient-based update to the attention weights, minimizing a semantic consistency loss derived from the LMM's output.
