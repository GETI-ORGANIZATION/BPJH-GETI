# Multimodal Data Fusion Agents for Resolving the Hubble Tension

## Idea Description
Development of an autonomous multimodal agent designed to ingest and simultaneously analyze heterogeneous cosmological datasets—specifically Planck, ACT, and SPT observations—to refine constraints on Early Modified Gravity (EMG) models as a solution to the Hubble tension.
